Methods

RRHO ideal-gas model with explicit electronic degeneracy; standard pressure fixed to 101 325 Pa. This page explains what is computed and why it’s accurate—while keeping implementation details private.

Partition functions

Translation (molar):
qtrans = ((2π m kBT)/h²)3/2 · (kBT/p)
Linear rotation (exact sum):
EJ/kB = θr J(J+1),  g = (2J+1)
qrot = (1/σ) · ΣJ=0..∞ g · e−EJ/(kBT)
Non-linear rotation (exact):
Rigid-rotor Hamiltonian diagonalized for spherical/symmetric/asymmetric tops; result divided by symmetry number σ.
Vibration (harmonic, per mode i):
qvib,i = 1 / (1 − e−θi/T)
Electronic:
qel = Σ gj e−θj/T (θ0=0 for ground state)

Notation: θi ≡ hνi/kB (vibrational temperature). Symmetry (σ) is applied wherever appropriate.

Thermodynamic relations

All terms use the same q they’re computed from:
S = R ln q + U/T,  U = R ⟨Ē⟩,  Cv = R·Var(Ē)/T²
Cp = Cv + R,  H = U + RT

Definitions: Ē ≡ E/kB (dimensionless). Assumptions: ideal gas; rigid rotor; harmonic oscillator; fundamentals listed with explicit degeneracies.

Scope & approximations

  • RRHO: Vibrational–rotational coupling and anharmonicity are neglected.
  • Ideal gas: Valid at low–moderate pressures; standard state p° = 101 325 Pa.
  • Electronic: Excited states included via degeneracies if thermally relevant.

Implementation details (e.g., numerical summation schemes, convergence controls) are intentionally omitted. We follow physically standard formulations and apply quality checks internally.
