Methods

RRHO ideal-gas model with explicit electronic degeneracy; standard pressure fixed to 101 325 Pa unless stated.

Partition functions

Translation (molar):
qtrans = ((2π m kBT)/h²)3/2 · (kBT/p)
Linear rotation (exact sum):
EJ/kB = θr J(J+1), g = (2J+1)
qrot = (1/σ) · ΣJ=0..∞ g · e−EJ/kBT
Non-linear rotation (exact):
Rigid-rotor Hamiltonian diagonalized for spherical/symmetric/asymmetric tops; result divided by symmetry number σ.
Vibration (harmonic, per mode i):
qvib,i = 1 / (1 − e−θi/T)
Electronics:
qel = Σ gj e−Θj/T (Θ0=0 for ground state)

Thermodynamic relations

All terms use the same q they’re computed from:
S = R ln q + U/T,   U = R ⟨Ē⟩,   Cv = R·Var(Ē)/T²
Cp = Cv + R,   H = U + RT

Assumptions: ideal gas; rigid rotor; harmonic oscillator; fundamentals listed with explicit degeneracies.
